## Система управления бизнесом (Упрощённая версия)

### О проекте
Это веб-приложение для управления командой внутри компании.
Упакован в докер-контейнер вместе с Redis и Postgres, реализовано автоматическое тестирование при запуске контейнеров. Реализовано как API, так и полноценное приложение с минимальным фронтендом 

### Основные возможности:

- Регистрация и управление аккаунтами пользователей

- Создание и администрирование команд (компаний)

- Назначение задач сотрудникам и отслеживание их выполнения

- Оценка работы сотрудников

- Планирование встреч и просмотр календаря событий

- Проект реализован как одностраничное MVP-приложение с минимальным фронтендом и понятной архитектурой, без микросервисов.

### Технологический стек
- FastAPI

- SQLAlchemy + Alembic для работы с БД и миграциями

- PostgreSQL - БД

- Pydantic - встроенная в FastAPI валидация

- fastapi-users - готовое решение по пользователями

- sqladmin - админка

- Jinja2 - шаблонизатор для HTML

- Uvicorn — ASGI сервер

- Docker

### Основная функциональность
#### 1. Пользователи
- Регистрация с email и паролем

- Авторизация и выход

- Роли: обычный пользователь, менеджер, администратор

- Обновление профиля

- Удаление аккаунта (без восстановления)

- Привязка к команде по коду - реализовано, но используется только team_id

#### 2. Команды
- Создание и администрирование команд (компаний)

- Добавление и удаление пользователей из команды

- Просмотр состава команды

- Назначение ролей внутри команды (менеджер, сотрудник)

#### 3. Задачи
- Создание задач руководителем

- Назначение исполнителей

- Описание, дедлайн, статус задачи

- Изменение и удаление задач

- Комментарии внутри задач (упрощённый чат)

- Статусы: открыто, в работе, выполнено

#### 4. Оценка работы
- Руководитель оценивает задачи (баллы от 1 до 5)

- Пользователь видит свои оценки

- Отображение средней оценки за выбранный период

#### 5. Встречи
- Планирование встреч с датой, временем и участниками

- Проверка конфликтов по времени

- Список встреч пользователя

- Возможность отмены встречи админом

#### 6. Календарь
- Отображение задач и встреч по дням

- Месячный и дневной вид в текстовом формате

- Автоматическое добавление задач и встреч в календарь

#### 7. Админ-панель
- реализовано с sqladmin
